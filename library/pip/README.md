# PIP

```cmd
  # to create venv
  python -m venv envName

  # to activate run
  envName/Scripts/activate.bat

  # to deactivate run
  deactivate

  # to install project dependency from requirements file
  pip install -r requirements.txt

  # to create a requirements file
  pip freeze > requirements.txt

  # to add system packages
  python -m venv envName --system-site-packages

  # to list local packages
  pip list --local

```
